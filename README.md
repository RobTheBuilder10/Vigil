# VIGIL

A mobile-first Christian Bible application designed for men seeking discipline, strength, and spiritual resilience.

**Scripture + Discipline + Brotherhood + Vigilance**

## Tech Stack

- **Frontend**: React Native (Expo) with TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **State**: React Context + AsyncStorage
- **Navigation**: React Navigation 6

## Core Features

1. **Bible Reader** - Multi-translation support (KJV, ESV, NKJV, NASB) with distraction-free mode
2. **Watchtower Daily** - Morning briefing with verse, interpretation, mission, and prayer
3. **The Vigil** - Daily discipline streak tracker (scripture, prayer, reflection, challenge)
4. **The Arsenal** - Verse memory system with flashcard drills and spaced repetition
5. **Battle Mode** - Spiritual first aid for anxiety, temptation, anger, and fear
6. **Brotherhood** - Community feed with prayer requests, testimonies, and small groups
7. **Missions** - 7/14/30 day guided challenges with badge rewards
8. **Prayer Journal** - Tactical notebook for logging and tracking prayers
9. **Scripture Audio** - Deep masculine narration with sleep timer
10. **Vigil Command** - Premium subscription tier

## Getting Started

```bash
npm install
npx expo start
```

## Project Structure

```
src/
├── components/     # Shared UI components
├── context/        # App state management
├── data/           # Bible data, sample verses, missions
├── navigation/     # React Navigation setup
├── screens/        # Feature screens
├── services/       # Storage, Supabase client
├── theme/          # Colors, typography, spacing
├── types/          # TypeScript type definitions
└── utils/          # Helper functions
```
